# IMDB-Movie-Recommender
An IMDb Movie Recommendation System that uses Selenium to scrape 2024 movie data, NLTK for text preprocessing, and Cosine Similarity to suggest similar movies through a Streamlit web interface
Data Extraction: Scraped live 2024 movie data from IMDb using Selenium.
Text Processing: Cleaned movie storylines using NLP (Tokenization & Lemmatization).
Vectorization: Converted text to numbers using TF-IDF.
Recommendation: Calculated movie similarity using Cosine Similarity.
UI: Interactive web app built with Streamlit.
